# Projeto-site-bolo
Site feito para atividade 

Muitos empreendedores locais perdem vendas e tempo devido à desorganização de pedidos recebidos de forma fragmentada pelo WhatsApp. Esta vitrine digital resolve esse problema ao profissionalizar a apresentação dos produtos e estruturar automaticamente o pedido do cliente. Ao centralizar a escolha de quantidades e o cálculo de valores em uma interface intuitiva e responsiva (Mobile First), reduzimos erros na comunicação e agilizamos o atendimento, permitindo que a empreendedora foque no que realmente importa: a produção artesanal.
